# 🧮 Python Calculator

## 📌 Description
This is a simple Python calculator that performs basic arithmetic operations like addition, subtraction, multiplication, and division.  
It is a beginner-friendly project to understand Python fundamentals such as input, conditions, and operators.

---

## ⚙️ Features
- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- ❌ Handles division by zero

---

## 🛠️ Technologies Used
- Python

---

## ▶️ How to Run
1. Make sure Python is installed on your system
2. Download or clone this repository
3. Open terminal or command prompt
4. Run the following command:


python calculator.py
